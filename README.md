# Spring Boot
### OAuth (Open Authorization)
인증과 권한을 동시에 부여하기 위한 프로토콜이다. </br>
예를 들어 자동차의 발렛키가 있다. </br>
발렛키는 트렁크를 열지못하고, 1km만 갈 수 있는 제한된 키이다. </br>

### Technique
- Google OAuth2 : Access Token과 Refres Toekn
- Google Calendar API : Service
- RestTemplate : http 통신
